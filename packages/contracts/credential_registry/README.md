# Credential Registry Contract

This package hosts the Soroban smart contract scaffold for the OpenCred credential registry.

The contract is intentionally kept as a stub so contributors can define:
- Soroban entrypoints for credential issuance and revocation
- data schema and storage structures
- contract authorization and access patterns

## Structure

- `src/lib.rs` — Rust contract entrypoint scaffold and comments.
- `Cargo.toml` — Rust package metadata.
